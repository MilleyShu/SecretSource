Infact, this works on ALL elements that you could possibly want to demonstrate, including plain old HTML!

SecretSource does NOT require jQuery. However, if you already have jQuery on your site, you can use the jQuery plugin of SecretSource. See the jQuery documentation for more information.

SecretSource is intentionally bare-bones. I have not tried to add every single use case possible to the code, as that would make the code much larger than it needs to be. Instead, consider this as a starting point that you can modify to suit your needs.
